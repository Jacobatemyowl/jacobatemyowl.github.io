Death Works in I.T.

Untitled Game Jam #89


This was the first one I've done solo in a while, and it was a lot of fun! Working with a theme of "Death is Useful," I chose to take it literally and took some inspiration from my job (not reflective, just inspired.) It was more technical than my last few games, so being able to stretch those muscles was good!

I saw a demo for a 2D character in a 3D background, so i wanted to give that a shot as well. Creating the interactions and movement was interesting, but I think it turned out well, overall. See the full itch.io submission here!

My game answers the question: "Should that be smoking?"
